# 问题

## 1.流式聊天为什么不用固定返回格式ChatResponse ?

流式返回无法加接口返回格式，response_model是给一次性返回JSON结构化数据的接口用的



## 2.with open中with的作用

打开文件后自动关闭，释放资源，计时报错也会自动执行f.close()



## 3.Few-shot是什么

少样本提示，在Prompt里举2-5个示例，让模型模仿风格



## 4.好的System Prompt 包括什么

角色（你是谁）+规则+输出格式（约束）+示例





# 总结

System Prompt很重要，可以用.txt文本文件存储，然后用一个程序对接调用所有的System Prompt
